CONHEÇA NOSSO PROJETO


JavaScript

JavaScript é uma linguagem de programação interpretada. É comumente usada para scripting no lado do cliente, mas que também pode ser executada em aplicações desktop ou no servidor.

Phaser

O Phaser é um framework open source para desenvolvimento de jogos em HTML5. Seu foco é criar jogos que rodam em navegadores tanto para mobile quanto para desktops.

Bootstrap

Bootstrap é o mais popular framework HTML, CSS, e JS para desenvolvimento de projetos responsivos e focado para dispositivos móveis na web.


X-MEN: TRAINING CENTER


Hoje em dia temos acessos a jogos em qualquer lugar e a qualquer hora. Muitos deles podem ser desenvolvidos em um tempo relativamente curto e, mesmo com sua simplicidade, conseguem alcançar os jogadores. Foi nessa perspectiva que desenvolvemos o X-Men: Training Center, nosso projeto final da disciplina de Lógica de Programação. Imaginamos um cenário no qual dois personagens principais, Ciclope e Gambit, são sujeitos a um treinamento, tendo em vista aprimorar seus poderes e testá-los em uma situação de perigo.

Nesse contexto, o jogo é dividido em dois níveis e o principal objetivo é salvar o Fera, que se encontra em uma prisão guardado por sentinelas. Na primeira fase o jogador deve capturar a chave da prisão e no segundo, enfrentar diversos sentinelas, liberar plataformas e utilizar a chave para liberar seu amigo. Esse projeto foi desenvolvido utilizando, principalmente, o framework open-source para o desenvolvimento de jogos Phaser e a linguagem de programação JavaScript.
